# Zipline
This script enables the user to place down two points and create a zipline between them. Have fun with it :)
## How to add a zipline in eden editor:
1. Place down 2 Game Logics.
2. Sync the two together.
3. Add the following line in the first Game Logic: [this,true] execVM "zipline.sqf";
4. Thats it! Start the game and have fun!
